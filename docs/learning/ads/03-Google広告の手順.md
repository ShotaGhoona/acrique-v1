# Google広告の手順

Google検索結果に広告を出す（リスティング広告）具体的な手順を解説する。

---

## 1. Google広告とは

### 広告の種類

```mermaid
graph TB
    GOOGLE[Google広告]
    GOOGLE --> SEARCH[検索広告<br>リスティング]
    GOOGLE --> DISPLAY[ディスプレイ広告<br>バナー]
    GOOGLE --> YOUTUBE[YouTube広告<br>動画]
    GOOGLE --> SHOPPING[ショッピング広告<br>商品画像付き]
    GOOGLE --> PMAX[P-MAX<br>全自動]

    SEARCH --> S1[ACRIQUEで使う<br>顕在層の刈り取り]
    style S1 fill:#c8e6c9
```

### 検索広告の表示位置

```
┌─────────────────────────────────────┐
│  Google 検索: アクリル板 オーダー 高級  │
├─────────────────────────────────────┤
│  [広告] ACRIQUE - 高級アクリル専門店   │  ← 検索広告（上部）
│  1cmの厚みが生む、圧倒的な存在感。     │
│  https://acrique.jp                  │
├─────────────────────────────────────┤
│  [広告] 競合A - アクリル板通販         │
├─────────────────────────────────────┤
│  自然検索結果1                        │  ← SEO（無料）
│  自然検索結果2                        │
│  ...                                 │
├─────────────────────────────────────┤
│  [広告] 競合B                         │  ← 検索広告（下部）
└─────────────────────────────────────┘
```

### Meta広告との違い

| 項目 | Meta広告 | Google検索広告 |
|------|---------|---------------|
| 表示タイミング | SNS閲覧中 | 検索した時 |
| ユーザーの状態 | 「なんとなく見てる」 | 「探してる」 |
| 訴求方法 | ビジュアル（動画/画像） | テキスト |
| 向いている層 | 潜在顧客（認知） | 顕在顧客（刈り取り） |

## 2. アカウント構造

### 3層構造

```mermaid
graph TB
    subgraph アカウント
        subgraph キャンペーン1: 検索_ブランド
            AG1[広告グループ1<br>アクリル板]
            AG2[広告グループ2<br>コレクションケース]
            AG1 --> KW1[キーワード群]
            AG1 --> AD1[広告文A]
            AG1 --> AD2[広告文B]
            AG2 --> KW2[キーワード群]
            AG2 --> AD3[広告文C]
        end
    end

    style AG1 fill:#e3f2fd
    style AG2 fill:#e3f2fd
```

### 各層で設定すること

| 層 | 設定項目 | 具体例 |
|----|---------|--------|
| **キャンペーン** | 予算・配信地域・入札戦略 | 「日1万円」「日本全国」「CV最大化」 |
| **広告グループ** | キーワード・広告文 | 「アクリル板 オーダー」「高級感を演出」 |
| **広告** | 見出し・説明文・URL | テキスト広告の内容 |

### Meta広告との構造比較

```mermaid
graph LR
    subgraph Meta広告
        M1[キャンペーン] --> M2[広告セット] --> M3[広告]
    end

    subgraph Google広告
        G1[キャンペーン] --> G2[広告グループ] --> G3[広告]
        G2 --> G4[キーワード]
    end
```

**違い**: Google広告には「キーワード」という概念がある

## 3. キーワードの基本

### キーワードとは

```mermaid
graph LR
    USER[ユーザー] -->|検索| KW[キーワード]
    KW -->|マッチ| AD[広告表示]
    AD -->|クリック| SITE[サイト]
```

「ユーザーが検索する言葉」と「設定したキーワード」が一致すると広告が表示される

### マッチタイプ

```mermaid
graph TB
    subgraph マッチタイプ
        A[完全一致<br>[アクリル板 オーダー]]
        B[フレーズ一致<br>"アクリル板 オーダー"]
        C[部分一致<br>アクリル板 オーダー]
    end

    A --> A1[その言葉ぴったり]
    B --> B1[その言葉を含む]
    C --> C1[関連する言葉も]
```

| マッチタイプ | 記法 | 検索例 | 表示される？ |
|-------------|------|--------|------------|
| **完全一致** | [アクリル板 オーダー] | アクリル板 オーダー | ◎ |
| | | アクリル板 オーダー 価格 | × |
| **フレーズ一致** | "アクリル板 オーダー" | アクリル板 オーダー 価格 | ◎ |
| | | オーダー アクリル板 | ◎ |
| | | アクリル 注文 | × |
| **部分一致** | アクリル板 オーダー | アクリル 特注 | ◎ |
| | | プラスチック板 カスタム | △（関連と判断されれば） |

### ACRIQUEのキーワード例

```mermaid
graph TB
    subgraph 商品系
        K1[アクリル板 オーダー]
        K2[アクリルケース 特注]
        K3[コレクションケース オーダーメイド]
    end

    subgraph 用途系
        K4[店舗 什器 アクリル]
        K5[フィギュア ケース 高級]
        K6[社名プレート 作成]
    end

    subgraph ブランド系
        K7[ACRIQUE]
        K8[アクリーク]
    end
```

| カテゴリ | キーワード例 | 想定CPC |
|---------|-------------|---------|
| 商品系 | アクリル板 オーダー | 100-200円 |
| 用途系 | 店舗什器 アクリル | 150-300円 |
| ブランド系 | ACRIQUE | 50-100円（競合少） |

### Step 3: コンバージョン設定

**やること**: 購入完了を計測できるようにする

```mermaid
sequenceDiagram
    participant User as ユーザー
    participant Google as Google広告
    participant Site as ECサイト

    User->>Google: 広告クリック
    Google->>Site: サイトへ誘導（GCLID付与）
    User->>Site: 商品購入
    Site->>Google: コンバージョン送信
    Google->>Google: 広告効果を計測
```

#### 設定方法

```
Google広告 → 目標 → コンバージョン → 新しいコンバージョンアクション
```

| 設定項目 | 選択/入力 |
|---------|----------|
| コンバージョンの種類 | ウェブサイト |
| カテゴリ | 購入 |
| コンバージョン名 | Purchase |
| 値 | 購入金額を動的に取得 |
| カウント方法 | 全件 |

### Step 7: 広告文作成

#### レスポンシブ検索広告（RSA）

```mermaid
graph TB
    subgraph 入力するもの
        H[見出し: 最大15個]
        D[説明文: 最大4個]
    end

    subgraph Googleが自動で組み合わせ
        AD1[見出し1 + 見出し2 + 見出し3<br>説明文1 + 説明文2]
        AD2[見出し3 + 見出し5 + 見出し1<br>説明文2 + 説明文1]
    end

    H --> AD1
    D --> AD1
    H --> AD2
    D --> AD2
```

#### 文字数制限

| 要素 | 文字数 | 個数 |
|------|--------|------|
| 見出し | 全角15文字（半角30文字） | 最大15個（最低3個） |
| 説明文 | 全角45文字（半角90文字） | 最大4個（最低2個） |

### Step 9: 審査と配信開始

```mermaid
sequenceDiagram
    participant You as あなた
    participant Google as Google
    participant User as ユーザー

    You->>Google: 広告を保存
    Google->>Google: 自動審査（通常1営業日）
    alt 審査OK
        Google->>User: 検索時に広告表示
    else 審査NG
        Google->>You: 不承認通知
        You->>You: 修正して再申請
    end
```

#### よくある審査落ちの理由

| 理由 | 対策 |
|------|------|
| 商標侵害 | 他社ブランド名を使わない |
| 誇大表現 | 「日本一」「絶対」を避ける |
| 句読点の過剰使用 | 「！！！」を使わない |
| キーワードと広告の不一致 | 関連性を高める |

## 5. 運用中の確認ポイント

### 検索語句レポート

**実際にどんな言葉で検索されて広告が表示されたか確認**

```
Google広告 → キーワード → 検索語句
```

```mermaid
graph TD
    A[検索語句レポート確認]
    A --> B{良いキーワード？}
    B -->|Yes| C[キーワードに追加]
    B -->|No| D[除外キーワードに追加]

    C --> E[例: アクリル板 厚み オーダー]
    D --> F[例: アクリル板 100均]
```

### 品質スコア

**広告の「成績表」。高いほど安く上位表示される**

```mermaid
graph LR
    subgraph 品質スコアの構成要素
        A[推定クリック率]
        B[広告の関連性]
        C[ランディングページの利便性]
    end

    A --> QS[品質スコア<br>1〜10]
    B --> QS
    C --> QS
```

| スコア | 評価 | アクション |
|--------|------|-----------|
| 8-10 | 良好 | 維持 |
| 5-7 | 普通 | 改善余地あり |
| 1-4 | 低い | 要改善 |

### 改善アクション

| 状況 | 原因 | アクション |
|------|------|-----------|
| インプレッション少ない | 入札が低い / キーワード少ない | 入札上げる / キーワード追加 |
| CTR低い（2%未満） | 広告文が弱い | 見出し・説明文を改善 |
| CPC高い | 競合が強い / 品質スコア低い | 品質スコア改善 |
| CVR低い | LP（着地ページ）が弱い | LP改善 |

### Meta広告との使い分け

```mermaid
graph LR
    subgraph 認知・興味喚起
        META[Meta広告<br>Instagram/Facebook]
    end

    subgraph 検討・刈り取り
        GOOGLE[Google検索広告]
    end

    META -->|興味を持った人が| GOOGLE
    GOOGLE -->|購入| CV[コンバージョン]
```

| 段階 | 広告 | 役割 |
|------|------|------|
| 認知 | Meta広告 | 「ACRIQUEって良さそう」と思わせる |
| 検討 | Google検索 | 「アクリル板 オーダー」で検索した人を獲得 |
