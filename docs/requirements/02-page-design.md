# 02. Page Design & UI/UX Requirements

## 1. サイト全体の雰囲気
- **ギャラリーのような佇まい:**
  - 商品自体が透明であるため、背景はシンプル（白ベースまたは黒ベース）にし、商品の「エッジ」と「影」を際立たせる写真を大きく配置する。
- **フォント:**
  - 欧文: 高級感のあるセリフ体 (e.g., Didot, Times New Roman系)
  - 和文: 洗練されたゴシック体 または 明朝体 (e.g., 游ゴシック, 筑紫明朝)

## 2. サイトマップ構成
1.  **TOPページ (Home)**
    - FV (First View): 1cm厚のアクリル断面の接写動画または画像 + キャッチコピー。
    - Navigation: [Shop] [Office] [You] の3入り口を明確に提示。
2.  **Category Pages (商品一覧)**
    - For Shop / For Office / For You の各ランディングページ。
    - 利用シーン（店内に置いた様子など）の写真を多用。
3.  **Product Detail (商品詳細 & オーダー画面)**
    - **重要機能:** 入稿データのアップロード機能、テキスト入力プレビュー（可能であれば）、サイズ選択によるリアルタイム見積もり。
4.  **About Us (ブランド/工場紹介)**
    - 「なぜ1cmなのか」「工場のレーザー技術」など、職人のこだわりを伝えるコンテンツ。
5.  **Gallery / Showcase**
    - ユーザーによる設置事例集（Instagram連携など）。
6.  **Contact / Bulk Order**
    - 法人向けの大口見積もりフォーム。

## 3. UI/UXの重要ポイント
- **「厚み」を伝える工夫:**
  - 商品写真は正面だけでなく、「斜め」や「側面」からのカットを必須とする。
  - 比較画像（3mmアクリル vs 1cmアクリル）などで価値を可視化する。
- **オーダーの簡略化:**
  - 「1個から作れる」手軽さを損なわないよう、入稿や文字指定のフローを極力シンプルにする。
  - 「デザインデータがない人」向けに、定型デザインテンプレートを用意する。

## 4. モバイル対応
- スマホでも「アクリルの輝き」が伝わる高解像度画像の最適化。
- SNS（Instagram/Pinterest）からの流入を意識し、縦型写真での表示崩れを防ぐ。
