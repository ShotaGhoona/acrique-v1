# 03. Item Lineup (Initial Phase)

初期リリース時にラインナップする主要商品リスト。

## A. For Shop (店舗・ディスプレイ)
> コンセプト: 「世界観を、置く。」

1.  **QR Code Cube (QRコード・キューブ)**
    - スペック: 50mm〜80mm角 / 10mm厚〜ブロック状
    - 加工: 裏面UV印刷 + 側面鏡面仕上げ
    - 用途: Instagram、決済用QRコード、Wi-Fi案内
2.  **Logo Cutout Object (ロゴ・カットアウト)**
    - スペック: 長辺100mm〜200mm / 10mm厚
    - 加工: レーザー切り抜き（ロゴ形状そのまま）
    - 用途: レジ横、受付サイン、撮影用プロップ
3.  **Luxury Price Tag (プライスカード)**
    - スペック: 名刺サイズ〜ハガキサイズ / 10mm厚
    - 加工: 表面レーザー彫刻 or UV印刷
    - 用途: ジュエリー、時計、高級商材のスペック表

## B. For Office (企業・B2B)
> コンセプト: 「信頼を、刻む。」

1.  **Floating Wall Sign (フローティング・ウォールサイン)**
    - スペック: A4〜A2サイズ / 5mm〜10mm厚
    - 付属品: 高級化粧ビス（ステンレス/真鍮）
    - 用途: エントランス社名看板、パーパスボード
2.  **Deal Tombstones (成約記念モニュメント)**
    - スペック: 自由形状 / 10mm厚〜積層
    - 加工: 内部封入風プリント、多層レイヤー構造
    - 用途: 上場記念、M&A成立記念、プロジェクト完了盾
3.  **Desk Name Plate (役員用ネームプレート)**
    - スペック: 横長 / 15mm〜20mm厚（または10mm積層）
    - デザイン: 幾何学的なカッティングを施したモダンデザイン
    - 用途: 社長室、役員デスク

## C. For You (個人・ギフト)
> コンセプト: 「瞬間を、閉じ込める。」

1.  **Trading Card Display (トレカ・アーキテクト)**
    - スペック: スタンダードサイズカード対応 / 3層構造（総厚15mm以上）
    - 特徴: UVカットアクリル使用、カード枠への装飾彫刻
    - 用途: ポケカ、遊戯王などの高額カード展示
2.  **Wedding Welcome Board (ウェディング・ボード)**
    - スペック: A3〜A2サイズ / 5mm〜10mm厚
    - デザイン: 透け感を活かしたタイポグラフィ、新郎新婦シルエット
    - 用途: 結婚式ウェルカムスペース、新居のインテリア
3.  **Baby Hand/Foot Print (手形足形アート)**
    - スペック: A4サイズ / 10mm厚
    - 加工: 実寸大データのレーザー彫刻（深く掘ることで立体感を出す）
    - 用途: 出産祝い、内祝い
