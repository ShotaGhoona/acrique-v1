# ACRIQUE 改善アイデア集

## エンジニア視点
- [ ] **3Dプレビュー機能** - 注文前にアクリル製品の光の屈折や厚みを3Dで確認できる
- [ ] **AIデザインアシスタント** - ロゴをアップロードすると最適なアクリル加工提案を自動生成
- [ ] **リアルタイム見積もり** - サイズ・厚み・数量を変えると即座に価格が変わるUI
- [ ] **AR試着機能** - スマホカメラで「自分の店舗に置いたらどう見えるか」を確認
- [ ] **Shopify/BASE連携** - 他ECプラットフォームとの在庫・注文同期
- [ ] **クーポンコード機能** - ECサイトにクーポン発行・適用機能を組み込む

## マーケティング視点
- [ ] **インテリア系インスタグラマーへのギフティング** - 映える製品なのでUGCを狙う
- [ ] **YouTuberへの無料配布** - 紹介動画で認知拡大
- [ ] **Pinterest広告** - ビジュアル重視のプラットフォームで訴求
- [ ] **結婚式場・ウェディングプランナーとの業務提携** - ウェルカムボード需要
- [ ] **美容室チェーンへのB2B営業** - 1店舗導入→全店舗展開を狙う
- [ ] **「推し活」特化のLP作成** - 推し活層はお金を惜しまない
- [ ] **Google検索広告** - 「アクリルスタンド オーダー」「店舗 什器 オリジナル」等

## デザイナー視点
- [ ] **施工事例ギャラリーの充実** - 実際に設置された写真が購買を後押し
- [ ] **素材サンプル郵送サービス** - 厚み・質感は実物を触らないと伝わらない
- [ ] **テンプレートデザイン提供** - デザインできない人向けに選ぶだけで完成するパターン

## 財務・経営視点
- [ ] **サブスクモデル** - 飲食店向け「季節メニュー板を年4回お届け」
- [ ] **法人向け掛け払い対応** - 請求書払い（Paid, NP掛け払い等）で企業導入ハードル下げる
- [ ] **ボリュームディスカウント明示** - 10個/50個/100個の価格表を見せて大口誘導

## カスタマーサクセス視点
- [ ] **制作過程の動画共有** - 「あなたの製品、今ここまでできました」で期待感向上
- [ ] **メンテナンスガイド同梱** - 長く使える＝高級品の証明
- [ ] **リピート割引** - 2回目以降10%OFF等でLTV向上

## 物流・オペレーション視点
- [ ] **店舗受け取りオプション** - 工場直接受け取りで送料無料＆傷リスク減
- [ ] **ギフトラッピング対応** - 記念品需要に応える
- [ ] **納期選択制** - 特急便（+料金）vs 通常便で顧客に選択肢

## PR・ブランディング視点
- [ ] **職人インタビュー記事** - 「誰が作っているか」でストーリー価値向上
- [ ] **環境配慮アピール** - アクリル端材のリサイクル取り組みをPR
- [ ] **受賞歴・メディア掲載の掲示** - 信頼性向上

## 異業種コラボ視点
- [ ] **建築事務所・インテリアデザイナーとの協業** - 設計段階から什器を提案
- [ ] **トレーディングカードショップとの提携** - カードディスプレイケース
- [ ] **葬儀社との提携** - メモリアルプレート需要
- [ ] **eスポーツチームへのスポンサー** - トロフィー提供で露出

## ユーザー心理視点
- [ ] **「他のお客様の声」レビュー機能** - 高単価商品は口コミが重要
- [ ] **比較表「vs 木製」「vs ガラス」** - なぜアクリルが優れているか可視化
- [ ] **後払い決済対応（Paidy等）** - 高単価でも買いやすく

## コミュニティ視点
- [ ] **SNSハッシュタグキャンペーン** - #ACRIQUE で投稿すると次回割引
- [ ] **お客様作品ギャラリー** - 購入者の使用例をサイトで紹介
- [ ] **推し活コミュニティとのコラボ企画** - 「推しのアクリルスタンド選手権」

---

## 優先度高（推奨）
この製品は「見ないとわからない」「触らないとわからない」高級品のため、以下を優先推奨：

1. **素材サンプル郵送サービス** - 実物の質感が最大の訴求ポイント
2. **施工事例ギャラリーの充実** - 導入イメージが購買決定を後押し
3. **法人向け掛け払い対応** - B2B顧客の導入ハードルを下げる
4. **レビュー機能** - 高単価商品は第三者の声が重要
