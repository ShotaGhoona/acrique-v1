## 画像の意図
高級ジュエリーショップのショーケース周りを捉え、アクリル製プライスタグやディスプレイ台がジュエリーと調和している様子を示す。Galleryのクローズアップより引いて、店舗空間としての高級感を伝える。

---

## 構図・イメージ詳細

### 被写体 (Subject)
- **メイン**: ジュエリーショーケースとその周辺
- **アクリル製品**: プライスタグ、リングスタンド、ネックレスディスプレイ
- **サブ**: ガラスショーケース、スポットライト、ベルベットトレイ

### カメラ設定
- **アングル**: やや斜め上から（ショーケースを見下ろす接客視点）
- **距離**: ミディアムショット（ショーケース1台分程度）
- **被写界深度**: やや浅め（手前のジュエリーにフォーカス、奥はソフトボケ）

### 照明
- **種類**: ショーケース用スポットライト + 間接照明
- **方向**: 上方からの集中照明
- **質感**: ハード（宝石の輝きを最大化）
- **効果**: ジュエリーとアクリルが共に輝く

### 色調・雰囲気
- **色温度**: クール（ダイヤモンドの輝きを引き立てる）
- **カラーパレット**: ブラック、ネイビー、ゴールド、クリア
- **コントラスト**: 高め（ラグジュアリー感）
- **全体の印象**: 高級感、信頼、洗練、宝石のような美しさ

### 構図ルール
- **配置**: ショーケースを画面中央に
- **奥行き**: 複数のジュエリーピースでレイヤー構成
- **光の演出**: スポットライトのハイライトを活かす
