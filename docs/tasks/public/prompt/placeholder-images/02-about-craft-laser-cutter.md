# About Craft - レーザーカッター画像

## 使用場所
- **ファイル**: `frontend/src/page-components/public/about/ui/sections/AboutCraftSection.tsx`
- **用途**: Craftsmanshipセクション、3枚並びの1枚目
- **アスペクト比**: 4/3
- **備考**: 「Laser Precision」を象徴。3枚並びなので他の2枚との統一感が必要

---

## 画像の意図
0.1mm単位の精密さを誇るレーザー加工技術を視覚的に伝える。最先端のテクノロジーと職人技の融合。ドラマチックでありながらクリーンな印象。

---

## 構図・イメージ詳細

### 被写体 (Subject)
- **メイン**: CO2レーザーカッターのヘッド部分とアクリル板
- **サブ**: レーザー光線（オレンジ/赤）、切断時のわずかな煙

### カメラ設定
- **アングル**: やや上から見下ろす斜め45度（レーザーヘッドとワークピースの両方が見える）
- **距離**: ミディアムクローズアップ（レーザーヘッドと加工点にフォーカス）
- **被写界深度**: 中程度（加工点はシャープ、背景の機械はややボケ）

### 照明
- **種類**: 環境光 + レーザー光自体
- **方向**: 上方からの柔らかい環境光
- **質感**: レーザー光がアクセントとして際立つ
- **効果**: レーザーの光点が最も明るく、周囲はやや暗め

### 色調・雰囲気
- **色温度**: クール基調 + オレンジのレーザー光（コントラスト）
- **背景**: クリーンな工房環境（ぼかして）
- **コントラスト**: 中〜高
- **全体の印象**: 精密、テクノロジー、プロフェッショナル

### 構図ルール
- **三分割法**: レーザー光点を交点に配置
- **視線誘導**: レーザーヘッド → 光点 → アクリル板
