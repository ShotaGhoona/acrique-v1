# Home Features - 大判対応 (Large Scale)

## 使用場所
- **ファイル**: `frontend/src/page-components/public/home/ui/sections/FeaturesSection.tsx`
- **用途**: 「選ばれる3つの理由」セクション、2枚目
- **アスペクト比**: 16/9
- **備考**: 右下に「A2 / 最大サイズ」のバッジがオーバーレイ。左右交互レイアウトでこの画像は右側配置

---

## 画像の意図
「A2サイズまでの大型加工が可能」を表現。
家庭用では不可能なスケール感、大判アクリルの迫力と美しさ。
エントランスサインや大型ディスプレイの可能性を感じさせる。

---

## 構図・イメージ詳細

### 被写体 (Subject)
- **メイン**: A2サイズ程度の大判アクリル板（立てかけ or 作業台上）
- **サブ**: スケール感を示す要素（作業台、定規、またはレーザーカッターのベッド）

### カメラ設定
- **アングル**: やや低めから見上げる（大きさを強調）
- **距離**: ミディアム〜ワイド（全体のサイズ感が伝わる）
- **被写界深度**: やや深め（全体像を把握できる）

### 照明
- **種類**: スタジオ照明 or 明るい工房の自然光
- **方向**: 上方または側面からの均一な光
- **質感**: クリーン、透明感を強調
- **効果**: 大判アクリルの透明度と存在感

### 色調・雰囲気
- **色温度**: ニュートラル
- **背景**: クリーンな工房環境（グレーの壁や作業台）
- **コントラスト**: 中程度
- **全体の印象**: プロフェッショナル、スケール感、可能性

### 構図ルール
- **縦のラインを強調**: 大判アクリルを立てることでスケール感を演出
- **三分割法**: アクリル板をフレームの2/3程度に
