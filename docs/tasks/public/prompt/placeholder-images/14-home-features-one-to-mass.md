# Home Features - 1個から量産まで (One to Mass)

## 使用場所
- **ファイル**: `frontend/src/page-components/public/home/ui/sections/FeaturesSection.tsx`
- **用途**: 「選ばれる3つの理由」セクション、3枚目
- **アスペクト比**: 16/9
- **備考**: 右下に「1個〜 / 最小ロット」のバッジがオーバーレイ

---

## 画像の意図
「1個からでも、100個単位でも対応可能」という柔軟性を表現。
小ロットのオーダーメイドから量産まで対応できる自社工場の強みを視覚化。

---

## 構図・イメージ詳細

### 被写体 (Subject)
- **メイン**: 同じ形状のアクリル製品が複数個、整然と並んでいる様子
- **サブ**: 製品のバリエーション（サイズ違い、形違いが少し見える程度）

### カメラ設定
- **アングル**: やや俯瞰（45度程度）、整列感が伝わる角度
- **距離**: ミディアム〜ワイド（複数個が画面に収まる）
- **被写界深度**: やや深め（手前から奥まで比較的シャープ）

### 照明
- **種類**: スタジオ照明
- **方向**: 上方からのソフトライト
- **質感**: 均一、クリーン
- **効果**: 各製品が均等に美しく見える

### 色調・雰囲気
- **色温度**: ニュートラル
- **背景**: 白または明るいグレー
- **コントラスト**: 中程度
- **全体の印象**: 量産品質、整然、プロフェッショナル

### 構図ルール
- **リピートパターン**: 同じ製品の繰り返しでリズム感
- **奥行き**: 手前から奥へと製品を配置し、深さを表現
- **三分割法**: 製品群を左2/3に配置
