# Home Features - 厚みと精密さ (Mass & Detail)

## 使用場所
- **ファイル**: `frontend/src/page-components/public/home/ui/sections/FeaturesSection.tsx`
- **用途**: 「選ばれる3つの理由」セクション、1枚目
- **アスペクト比**: 16/9
- **備考**: 右下に「10mm / 標準厚み」のバッジがオーバーレイ

---

## 画像の意図
「1cmの厚みを標準に。自立する重厚感と、光の屈折が生み出す美しさ」を表現。
薄いアクリルとの違いを視覚的に伝える。厚みがあるからこそ生まれる「塊」としての存在感。

---

## 構図・イメージ詳細

### 被写体 (Subject)
- **メイン**: 10mm厚のアクリルブロックの断面・側面（厚みが明確にわかる角度）
- **サブ**: 断面を透過する光、エッジの鏡面反射

### カメラ設定
- **アングル**: ほぼ水平、やや斜めから（厚みが強調される角度）
- **距離**: クローズアップ〜ミディアム（断面のディテールと全体像の両方）
- **被写界深度**: 中程度（断面はシャープ、背景は適度にボケ）

### 照明
- **種類**: スタジオ照明
- **方向**: 側面からのサイドライト（エッジを光らせる）
- **質感**: シャープ、エッジが際立つ
- **効果**: 厚みのある断面が「光の壁」のように見える

### 色調・雰囲気
- **色温度**: ニュートラル
- **背景**: ソフトなグレーまたは白グラデーション
- **コントラスト**: 中〜高（エッジの存在感を強調）
- **全体の印象**: 精密、重厚、高品質

### 構図ルール
- **水平基調**: 16/9のワイドフォーマットを活かした横長構図
- **三分割法**: アクリルブロックを左2/3に配置（右下バッジ用に空間確保）
