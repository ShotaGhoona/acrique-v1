# ACRIQUE 商品分類戦略アドバイス

## 1. 現状の課題整理

現在、3つの分類軸が混在しており、以下の混乱が生じている：

| 分類軸 | 内容 | 用途 |
|--------|------|------|
| **ターゲット軸** | Shop / Office / You | マーケティング・導線設計 |
| **形状軸** | Signature / Standard / Free-cut / Structure | 製品ラインナップ |
| **製作軸** | 既製品 / セミオーダー / フルオーダー | 製作フロー・価格設定 |

**問題点：**
- 同じ商品が複数のターゲットに該当する（The Canvasは Shop にも You にも売れる）
- 形状と製作方式が1対1で対応しない（The Canvas は既製品にもフルオーダーにもなりうる）
- ECサイトのカテゴリ構造と製品構造がズレている

---

## 2. 推奨：分類軸の役割分離

### 結論：「見せ方」と「作り方」を完全に分ける

```
┌─────────────────────────────────────────────────────────┐
│  【顧客に見せる軸】                                      │
│   ├─ ターゲット軸（Shop/Office/You）→ サイト導線        │
│   └─ 形状軸（Canvas/Silhouette等）→ 商品名・ブランド    │
├─────────────────────────────────────────────────────────┤
│  【内部管理軸】                                          │
│   └─ 製作軸（既製品/セミ/フル）→ 価格・納期・入稿フロー │
└─────────────────────────────────────────────────────────┘
```

---

## 3. 各軸の再定義

### 3-1. ターゲット軸（マーケティング用）

**目的：** 顧客が「自分ごと」として認識できる入口を作る

| ターゲット | 訴求ポイント | 主な商品（例） |
|-----------|-------------|---------------|
| **For Shop** | 「既製品POPでは安っぽい」を解決 | メニュースタンド、QRスタンド、A看板 |
| **For Office** | 「特注は発注ハードルが高い」を解決 | ロゴサイン、表彰盾、会議室プレート |
| **For You** | 「劣化しない記念品が欲しい」を解決 | 推しパネル、結婚証明書、ペット写真 |

**注意：** これは「商品カテゴリ」ではなく「入口の切り分け」。同じ The Canvas でも、For Shop では「メニュー写真を美しく」、For You では「推しを永遠に」と訴求を変える。

|---------|--------|---------|--------|
| **Signature** | The Canvas | 定型四角（A2-A5） | 低 |
| **Standard** | The Canvas Ratio | 変形四角（自由比率） | 低 |

| **Structure** | The A Stand | A看板 | 中 |

---

### 3-3. 製作軸（オペレーション用）

**目的：** 価格・納期・入稿フローの決定

| 製作タイプ | 定義 | 特徴 |
|-----------|------|------|
| **Ready-made（既製品）** | 完成品として在庫。注文後即発送 | 価格固定、最短納期、入稿不要 |
| **Semi-order（セミオーダー）** | デザインは固定、一部だけ差し替え | QRコードURL、テキスト名入れ等 |
| **Full-order（フルオーダー）** | 顧客の画像/ロゴを入稿して製作 | 価格変動あり、要入稿審査 |

## 5. 販売戦略への提言

### 5-1. 価格設計

```
┌─────────────────────────────────────────────────────────┐
│  基本価格 ＝ 形状基本価格 ＋ サイズ差分 ＋ 製作タイプ差分  │
└─────────────────────────────────────────────────────────┘
```

| 製作タイプ | 価格係数 | 理由 |
|-----------|---------|------|
| Ready-made | 1.0x | 基準価格 |
| Semi-order | 1.2x〜1.5x | データ処理工数 |
| Full-order | 1.5x〜2.0x | 入稿審査・調整工数 |

**提案：** Semi-order を積極展開し、「手軽さ × プレミアム感」の両立を狙う。Full-order は利益率が高いが、入稿審査のオペレーション負荷も高い。

---

### 5-2. 導線設計（ECサイト）

```
【現状のサイト構造】
  /shop    → Shop向け商品一覧
  /office  → Office向け商品一覧
  /you     → You向け商品一覧

【推奨】
  ターゲット別入口は維持しつつ、
  同じ商品を複数カテゴリに表示（タグベース）
```

**例：** The Canvas は `/shop`（メニュー写真）、`/you`（推しパネル）両方に表示。商品詳細ページは共通で、訴求文言だけ出し分け。

---

### 5-3. マーケティング優先順位

**Phase 1: For You（個人）を最優先**
- 理由：SNS拡散力が高い（推し活、結婚式）
- 単価は低いが、口コミ・UGCが期待できる
- 主力商品：The Canvas、The Silhouette

**Phase 2: For Shop（店舗）**
- 理由：リピート購入が見込める（メニュー更新、新店舗）
- 主力商品：The A Stand、QRスタンド系

**Phase 3: For Office（法人）**
- 理由：単価は高いが、意思決定に時間がかかる
- 主力商品：The Logo、表彰盾系

## 6. 商品ラインナップの優先順位

### MVP（最初にリリースすべき商品）

| 優先度 | 商品 | 製作タイプ | 理由 |
|--------|------|-----------|------|
| ★★★ | **The Canvas** | Full-order | 最も汎用的、全ターゲットに訴求可能 |
| ★★★ | **The Canvas Round** | Full-order | 形状の差別化、インスタ映え |
| ★★☆ | **The A Stand** | Semi-order | 店舗向けの定番ニーズ |
| ★★☆ | **The Silhouette** | Full-order | 推し活市場で差別化 |
| ★☆☆ | **The Logo** | Full-order | 法人向け、要営業 |

### 後回し（オペレーション安定後）

- The Typography（難易度最高、再入稿リスク大）
- The Canvas Puzzle（ニッチ、結婚シーズン限定）
- The Square Stand（大型、配送コスト高）

## 7. 命名規則の提案

現状の「The Canvas Ratio」等は長い。以下のルールを推奨：

```
【ブランド名】＋【形状】＋【サイズ/オプション】

例：
  ACRIQUE Canvas A3        （定型四角 A3サイズ）
  ACRIQUE Round M          （円形 Mサイズ）
  ACRIQUE Silhouette       （輪郭切り抜き）
  ACRIQUE A-Stand Single   （A看板 片面）
```

## 8. 次のアクション

1. **商品マスタの設計**
   - 形状 × 製作タイプ × サイズ の組み合わせを洗い出す
   - 各組み合わせの価格を決定

2. **入稿フローの整理**
   - Semi-order：何を入力させるか（URL? テキスト?）
   - Full-order：どのフォーマットを受け付けるか

3. **撮影・コンテンツ制作**
   - 各商品の使用シーン写真
   - ターゲット別の訴求コピー

## 9. 質問事項（要確認）

1. **既製品（Ready-made）は展開するか？**
   - サンプル販売 or デモ品のみ？
   - 在庫リスクをどこまで許容？

2. **Semi-order の範囲**
   - QRコードは「URL入力」で自動生成？それとも「画像入稿」？
   - 名入れテキストのフォントは固定？選択式？

3. **価格帯の目安**
   - The Canvas A4 Full-order で想定価格は？
   - 競合比較でのポジショニングは？

4. **ターゲット別ページの位置づけ**
   - 完全に分離（別カテゴリ）？
   - 同じ商品を複数表示（タグベース）？
