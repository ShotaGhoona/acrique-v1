# ACRIQUE Business Presentation

Slidevを使用したACRIQUEビジネス説明用スライドです。

## 起動方法

```bash
cd slides
npm run dev
```

ブラウザで http://localhost:3030 を開くとスライドが表示されます。

## スライド構成

1. **タイトル** - ACRIQUE紹介
2. **ブランドコンセプト** - 「透明な素材を、重厚なインテリアへ昇華させる」
3. **Core Value** - 3つの強み（1cm厚み、1個から対応、A2サイズ）
4. **ターゲット** - For Shop / For Office / For You
5. **商品紹介** - 各カテゴリの主要商品とユースケース
6. **人気商品ピックアップ**
7. **注文フロー** - 商品選択から納品まで
8. **品質へのこだわり** - 素材と加工技術
9. **Why ACRIQUE** - 選ばれる理由
10. **課題解決** - ACRIQUEが解決する課題
11. **CTA** - お問い合わせ・ご注文案内

## エクスポート

PDFとしてエクスポートする場合:

```bash
npm run export
```

## カスタマイズ

- `slides.md` を編集してスライド内容を変更
- デザインシステム準拠のカラー:
  - Primary: `#171717` (ほぼ黒)
  - Accent: `#0ea5e9` (Ice Blue)
  - Background: `#ffffff` (白)

## 操作方法

- `→` / `Space`: 次のスライド
- `←`: 前のスライド
- `o`: スライド一覧
- `p`: プレゼンターモード
- `f`: フルスクリーン
